# create-the-chair

Scaffold a new self-hostable blog powered by [The Chair](https://github.com/twinprime19/blog).

## Usage

```bash
npx create-the-chair my-blog
cd my-blog
npm start
```

Your blog will be at `http://localhost:1911`.

**Requires:** Node.js >= 18 and git

See the [main repo](https://github.com/twinprime19/blog) for full documentation.
